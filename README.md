# News Scallion

## Overview
News Scallion is a web tool that summarizes real news articles with a humorous twist, generating AI-driven content for laughs and criticism.

## Features
- AI-generated summaries of news articles
- User registration and login
- Messaging system for users
- Admin panel for managing content

## Installation
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the server with `node server.js`.

## Usage
Visit `http://localhost:3000` to use the site.
